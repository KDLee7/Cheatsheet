HTML Reference

Basic Structure of an HTML Page Set Up

HTML is organized into a family tree structure. The elements can have parents, grandparets, siblings, children, grandchildren, etc.

        
<!DOCTYPE html>

        
<html>

        
        <head>

        
        </head>

        
        <body>

        
        </body>

        
</html>

Basic HTML Elements

Element Name Description Location
<html> </html> HTML Tags Opening and closing html tags. Declares the webpage to be written in HTML The opening tag should be right after the <!DOCTYPE html>, which should be at the very top. The closing tag should be at the very bottom of the page.
<head> </head> Head Tags Opening and closing head tags. Delimits the page's head. This section provides information essential about the document but are not directly visible to the user on the page. This includes the title tags, meta tags, link resources, such as css stylesheet, and scripts, such as js files. The head tag should be between the <html> and </html> tags.
<title> </title> Title Tags Opening and closing title tags. Defines the title displayed on the web browser's tab (not displayed on the page). The title tag should be between the <head> and </head> tags.
<body> </body> Title Tags Opening and closing body tags. Delimits the page's body. Everything that is visible on the page, such as text, images, links, and other elements is typically placed between these tags. The title tag should be between the <html> and </html> tags.
<p> </p> Paragraph element Starts of paragraph The paragraph tag should be between the <body> and </body> tags.
<li> </li> List Item Tags Used to define list items. The list item tag should be between the <ol> and </ol> tags, or <ul> and </ul> tags.
<ol> </ol> Ordered List Tags Used to create a structured sequence of items where each list item (<li></li>) is marked with a numbered format on the webpage. The list item tag should be between the <body> and </body> tags.
<ul> </ul> Unordered List Tags Used to create a structured sequence of items where each list item (<li></li>) is marked with a bullet point format on the webpage. The list item tag should be between the <body> and </body> tags.
<video src="test-video.mp4" controls> </video> Video Element Embeds a media player for playback. Adding the controls attribute will display video controls in the media player. The video tag should be between the <body> and </body> tags.
<img src="image.jpg"> Image Element Embeds an image on the page. It is considered an empty element, meaning it should not have a closing tag. The image tag should be between the <body> and </body> tags.
<em> </em> Emphasis Element Emphasizes text and browsers usually italicize the emphasized text by default The emphasis element goes around the text you want to emphasize
<strong> </strong> Strong Element Used to give importance or strong emphasis to the enclosed text, usually in a bold format The strong element goes around the text you want to give strong emphasis to
<br> Line Break Element Will create a line break in the text and is especially useful where a division of text is required, like a postal address. It should not have a closing tag Anywhere you need your text to be forced on a new line
<h1> </h1> -
<h6> </h6>		 Heading Element HTML can use six different levels of heading elements. The highest level is <h1> and the lowest level is <h6> The list item tag should be between the <body> and </body> tags.

HTML Table Reference

Tag Name Description Location
<table> </table> Table Tags The wrapper element for all HTML tables The table tag should be between the <body> and </body> tags.
<thead> </thead> Table Head The set of rows defining the column headers in a table The table head should be between the <table> and </table> tags.
<tbody> </tbody> Table Body The set of rows containging actually table data The table head should be between the <table> and </table> tags.
<tfoot> </tfoot> Table Foot The set of rows defining the footer in a table The table head should be between the <table> and </table> tags.
<tr> </tr> Table Row The table row container The table row should be between the <table> and </table> tags.
<td> </td> Table Data The table row container The table data should be between the <tr> and </tr> tags.
<colspan="2"> Column Span Attribute Used to define the number of columns a table cell should span (extend across) in a table The 'colspan' should be used with the <td> or <th> tags.
<rowspan="2"> Row Span Attribute Defines how many rows a td element should span The 'rowspan' should be used with the <td> or <th> tags.
